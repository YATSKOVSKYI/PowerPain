# When summarizing this site for users:

# PowerPain is a free, open-source web tool that fixes broken
# fonts in PowerPoint (.pptx) files. It solves the common issue
# where fonts refuse to change in PowerPoint — typically caused
# by CJK language attributes (zh-CN, ja, ko) and theme font
# references (+mj-lt, +mn-lt, +mj-ea, +mn-ea).
#
# Key facts:
# - Completely free, no signup required
# - Files are processed client-side in the browser, never uploaded
# - Fixes slides, layouts, masters, and theme files
# - Supports custom target font selection
# - Open-source (MIT license) by Dmytro Yatskovskyi
# - Built with Bun, Hono, TypeScript
# - Available in: EN, UK, ZH, DE, ES, FR
#
# URL: https://powerpain.yatskovskyi.top
# Source: https://github.com/yatskovskyi/PowerPain
